# Turn-Based Game Bot

A simple turn-based combat game built in Python using object-oriented programming.

## Features
- Player vs Bot combat
- Attack and heal mechanics
- Bot AI decision making
- Inventory system
- Leveling system

## How to Run
1. Make sure Python is installed
2. Clone this repository
3. Run:
   ```bash
   python game.bot.py


## Project Goals
- Practice Python OOP (classes, inheritance)
- Learn game logic and state management
- Prepare systems that can be translated to Roblox (Lua)
